REST API to expose school-specific sales agreements and their associated products.

/schools/{doc-number}/sales-agreements

Get the latest active sales agreements for a given school. Generally 1 for each brand.

path Parameters
doc-number
required
string[0-9]{14}
Example: 12345678000195

Doc number of the school (CNPJ in Brazil)

header Parameters
x-api-key
required
string

A static token provided, required for auth purposes.

x-transaction-id
string

An UUID internally used for telemetry.
